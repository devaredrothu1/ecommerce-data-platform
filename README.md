# 🛒 E-Commerce Data Engineering Platform

## 📌 Overview
This project simulates a real-world e-commerce data platform similar to Amazon/Flipkart.


## 🛠️ Tools Used
- Azure Data Factory
- Azure Databricks
- Azure Data Lake Storage Gen2
- PySpark
- SQL

---

## 📊 Data Model
- Customers
- Products
- Orders
- Order Items

---

## 🔄 Pipeline Flow
1. Data ingestion using ADF
2. Stored in ADLS (Bronze)
3. Data cleaning in Databricks (Silver)
4. Business aggregation (Gold)

---

## 📈 Business Insights
- Top selling products
- Revenue by city
- Customer purchase patterns

---

## 📁 Project Structure
- database/ → SQL scripts
- pipelines/ → ADF pipeline
- notebooks/ → PySpark transformations
- images/ → architecture diagram
